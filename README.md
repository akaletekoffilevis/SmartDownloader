# SmartDownloader

A tool for downloading videos from various platforms.

## Features

- Download videos from YouTube, Vimeo, and other platforms
- Convert videos to different formats
- Download playlists and channels
- Extract audio from videos

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py [URL]
```
